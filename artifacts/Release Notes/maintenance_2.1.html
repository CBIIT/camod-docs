What's new in the upgraded caMOD?

  • Support of zebrafish models
  • The ability to save search queries, as well as the ability to configure your search results.
  • Enhanced searching capabilities.
 
Summary of new features
Added Features How to use the new feature?
Keyword search highlighting When using the keyword search, the keyword is highlighted in yellow in the search results pages
Configurable search results A logged in user can customize their search results. They can configure which columns to display and how results are displayed on each screen.
Saving search queries for reuse A logged in user can access a listing of their previously run search queries and can save particularly interesting queries for later reuse.
Auto completion of search fields During searching, a user can type in text in the model characteristics, organ, diagnosis and NSC number fields and the application will prompt the user with appropriate entries.
