What's new in the upgraded caMOD?

  • New user interface for easier model submission
  • Numerous new features
  • caMOD 2.0 is now caBIG silver level compliant
 
Summary of new features
Added Features How to use the new feature?
Utilization of NCI Thesaurus caMOD 2.0 utilizes the vocabulary services of NCI Thesaurus. The mouse anatomy and disease vocabularies have been updated by the Pathology working group of the Mouse Models of Human Cancers Consortium (MMHCC).
Utilization of CSM caMOD 2.0 utilizes the Common Security Module (CSM) for user authentication.
Submission - previously entered models After the user logs in, his/her previously entered models and models in which he/she is listed as the Principal Investigator are shown.
Submission - Model Characteristics The scope of the cancer models database was expanded to include tool mice. Tool mice or tool animals are strains that can be used to generate cancer models and do not normally develop tumors. Tool mice are commonly used to generate conditional knock out animals. Examples for tool strains are Cre-expressing strains or strains that carry one or more floxed alleles.
Submission - Genetic Description
  • A field was added to all pages in this part of the application to allow the submitter to enter the MGI (Mouse Genome Informatics) number which will enable the retrieval of additional information from the informatics resources at the Jackson Laboratory.
  • The Targeted Modification page and the Induced Mutation page allow the user to enter the Gene Entrez ID for the affected gene. This will enable us to provide users with additional information about the gene(s).
  • The Induced Mutation page contains a field for Gene Entrez ID for the affected gene. If a chemical triggered the mutation, the user has the option to enter the CAS number for the chemical that was used.
  • A submission page for Spontaneous Mutations was added.
  • The submission pages for Induced and Spontaneous Mutations contain fields to gather information about the genetic changes that occurred in the models as well as the methods of observation used to detect these changes.
Submission - Carcinogenic Intervention
  • A page for entering information on Gene Delivery was added to this part of the application.
  • All pages of the Carcinogenic Interventions part were updated to include fields for gender, age at treatment, and administrative route (if applicable).
  • The Chemical / Drug page allows the user to enter the NSC and/or CAS number of the chemical used in this experiment.
Submission - Publication The user is asked to indicate if the publication is the one in which the model was reported for the first time. Eventually this feature will be used to link to records in the Mouse Tumor Biology Database at the Jackson Laboratory.
Submission - Histopathology
  • Users can submit information about clinical markers and their values which are found in or are associated with specific tumor types.
  • A field for describing the methods of observation / detection of genetic changes in the tumors was added.
  • Users can enter data generated from other species, in particular humans, in the "Comparative Data" field.
Submission - Therapeutic Approaches
  • To better identify the drug, users can enter the NSC and the CAS number for the compound.
  • More detailed information about the therapy can be archived. The following fields have been added: age at treatment, gender, dosage and administrative route.
  • The results of experiments can be described in the biomarker and tumor response fields.
Submission - Microarray Data Micrroarray data are stored in MAGE-compliant caArray application instead of the Gene Expression Data Portal (GEDP).
Submission - Images The text field for entering the staining method was replaced by a controlled vocabulary.
Submission - Xenograft Xenograft models can now be entered in a specifically designated part of the application. The Xenograft page adds the following fields: ATCC number, cell amount and graft type.
Submission - Model Availability Users can now enter information about the availability of the animal model from multiple sources. They can also indicate if the tool strains, used to generate the models, are available from multiple sources. The users can make multiple entries per model.
Administrative Users who have been assigned the roles of coordinator, screener, and editor will be presented with a to-do list of tasks when they access the Administrative section, following log in.
Search - Simple Search The simple search was expanded with a keyword search option.
Search - Advanced Search The advanced search now includes a query option for drugs or compounds used in therapeutic experiments, an option to query for models that develop metastases, and a keyword search.
Search - Search Detail Pages The therapeutic approaches pages include information about drug screens in yeast and in-vivo, as well as clinical trials.
Search - Navigation Only search detail pages that contain data will have active links on the left side menu bar.
Search - Drug Screening The user can query for results of drug testing in yeast strains, xenograft and allograft mouse models (titled for this purpose as in-vivo models), pre-clinical models, and humans. The tests in yeast and in-vivo models were performed by the Developmental Therapeutics Program of the National Cancer Institute. Information about pre-clinical models is retrieved from caMOD, and human data are retrieved via caBIO from clinical trials.
