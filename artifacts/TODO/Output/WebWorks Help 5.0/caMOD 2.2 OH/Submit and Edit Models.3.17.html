Submitting and Editing Models > Carcinogenic Interventions
Carcinogenic Interventions
Chemical/Drug
To enter or edit a chemical or drug for an animal model:
1.
Click Carcinogenic Interventions from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Chemical/Drug red hypertext link. This opens the Chemical/Drug form.
3.
Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.
Chemical/Drug Fields
Description
*Chemical/Drug
Select the Chemical or Drug from the drop-down list. If the appropriate Chemical/Drug is not listed, select Other from the drop-down list and enter the Other Chemical/Drug in the text field under the list.
Other Chemical/Drug
Enter Chemical/Drug that is not listed in the drop-down list. Note: This field is only writable after you select Other in the previous field.
Dose
Enter the dose of the chemical agent or drug given. Select the dosage units from the drop-down menu.
 
Note: If the dose unit you are looking for is not available in the drop down list, manually enter the dose unit together with the amount in the text field. When you return to the page, your dose unit entry that you selected or entered manually will be displayed in the text input field. Should you decide to update the dose unit information, you must delete the current dose unit entry from the text field before selecting or entering the new information. NSC number
Enter the NSC number in the text box. The NSC number is a unique drug identifier issued by the Nomenclature Standards Committee (NSC) of the U.S Food and Drug Administration.
•
If you don't know the number, click the Find NSC# button. The button opens a page at the Developmental Therapeutics Program (DTP) at NCI which allows you to find the NSC number for a specific drug.
•
On the DTP page, select Chemical Name Starts or Chemical Name Number. (By default, the radio button is set to find the NSC number.)
•
Enter text in the Search Parameters field, and click Submit.
•
Once you have found the NSC number, copy the number from the field onto the field on the caMOD Chemical/Drug page.
CAS number
Enter the Chemical Abstracts Service (CAS) number for a chemical or drug. Use the provided hypertext link (Find CAS#) to find the CAS number. Copy the number and paste into the field on the caMOD Chemical/Drug page.
Note: You can also retrieve the CAS number from the DTP page described in the previous row of this table. It will display with the NSC number on the Chemical/Drug page.
Administrative Route
Select the course in which a drug was administered in order to reach the site of action in the body. If the appropriate route is not listed, select Other from the drop-down list and enter the Other Administrative Route in the text field under the list.
Other Administrative Route
Enter Administrative Route that is not listed in the drop-down list. Note: This field is only writable after you select Other in the previous field.
Treatment Regimen
Enter times and intervals in which the treatment was administered.
Age at Treatment
Enter age information in the text box and from the drop-down list.
Note: If the time unit you are looking for is not available in the drop down list, manually enter the time unit together with the number in the text field. When you return to the page, your time unit entry that you selected or entered manually will be displayed in the text input field. Should you decide to update the unit information, you must delete the current time unit entry from the text field before selecting or entering the new information. Should you fail to delete first, the new time unit entry will be added to the old entry.
Gender
Select gender from the drop-down list.
4.
Click the Submit button to execute the submission of the model information.
5.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
