Entering or Editing siRNA Information
To enter or edit the siRNA information for an animal model:
1.
Click Transient Interference from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter siRNA red hypertext link. This opens the siRNA form.
3.
Enter or select the appropriate information about your siRNA molecule in the text entry fields. Items with an asterisk are required fields.
4.
Fill in correct fields and describe.

siRNA Fields
Description
Source
Select the source of the siRNA in the drop-down menu. Options are companies that make siRNAs: Gene Tool; Biosystem
If the appropriate source is not listed, select Other from the drop-down list and enter the Other Source in the corresponding text field.
Other Source
Enter the siRNA source that is not listed in the drop-down list. Note: This field is only writable after you select Other in the previous field.
Type
Select from the drop-down menu the siRNA type, Custom or Standard.
Sequence Direction
Select the sequence direction in the drop-down menu: 5' - 3' or 3' - 5'.
Â 
*Targeted Region
Enter the Targeted Region (site or locus) affected by the siRNA in the text box.
Note: After the data have been submitted, the siRNA is identified under the Transient Interference > siRNA category by the term used in this text box.
Concentration
Enter the Concentration of the sample in value and measurement units.
Delivery Method
Select from the drop-down menu the method used to deliver the siRNAs. If the appropriate method is not listed, select Other from the drop-down list and enter the Other Delivery Method in the corresponding text field.
Other Delivery Method
Enter Other Delivery Method that is not listed in the drop-down list.
Visualization Ligands
siRNAs can be combined with fluorescent ligands for visualization purposes. For example, fluorescein isothiocyanate is a visualization ligand. Enter the visualization ligand used, if appropriate.
Other Visualization Ligands
Enter Other Visualization Ligand that is not listed in the drop-down list.
Comment
Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.
5.
Click the Submit button to execute the submission of the model information.
6.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
