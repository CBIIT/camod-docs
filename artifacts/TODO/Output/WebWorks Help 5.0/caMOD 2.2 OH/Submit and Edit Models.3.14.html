Targeted Modification
To enter or edit the Targeted Modification information for an animal model:
1.
Click Genetic Description from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Targeted Modification red hypertext link. This opens the Tar­geted Modification form.
3.
Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.
 
Targeted Modification Text Fields
Description
*Targeted Gene/Locus
Define the gene or region in the genome that are altered by the substitution of a DNA construct.
*Type of Modification
Select one or more mutation types from the drop-down menu.
Examples:
•
Null mod--a type of modification in which the altered gene product lacks the molecular function of the wild-type gene
•
Amino acid--modification that alters one or more amino acids
•
Insertion--insertion of one or more residues in the molecular structure
•
Missense--a codon specifying one amino acid altered so as to specify a different mino acid
Note: For this field, you can select multiple choices. If one of your choices is not listed, select Other from the drop-down list and enter the Other Modification Type in the text field under the list.
Other Modification Type
Describe any appropriate type of modification. Note: This field is only writable after you select Other in the previous field.
Gene ID (Entrez)
Enter gene Entrez ID number. Click the Find Gene ID button for a link to Entrez where you can locate the ID.
Genetic Background
Enter the appropriate information regarding the genetic background.
ES Cell Line: The name of the Embryonic Stem Cell line that was used to insert the DNA construct with the alteration. When incorporated into chimeric mouse embryos, they are totipotent: they can differentiate into any cell type in the mouse.
Blastocyst: Strain background of the blastocysts in which the modification-carrying ES cells were inserted.
Conditional
Select the Conditional or Not Conditional radio button.
Conditional Description
Where appropriate, enter the description of the strategy to generate a conditional odification.
Conditional modification is most commonly used to represent Cre-mediated excision of genomic sequences flanked by IoxP sites. The excised genotype, which is often associated with a phenotype, is dependent (or conditional) on the presence of the Cre-expressing construct.
MGI Number
Enter the Mouse Genome Informatics (MGI) ID for this mutation. Note: You can copy and paste the entire MGI ID (including “MGI”) into this field.
Click Find MGI# to open the MGI website where you can learn more about the MGI, including the MGI# for your model.
Comment
Enter any information that did not fit or was not suitable for other fields on the page.
4.
Under the gray line on the Targeted Modification form, select or enter the requested information about the model construct, if you decide to upload a map of the construct. For more information, see Model Construct.
5.
Click the Submit button to execute the submission of the model information.
6.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
