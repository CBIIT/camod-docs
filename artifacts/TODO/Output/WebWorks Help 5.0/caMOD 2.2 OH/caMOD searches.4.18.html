Launching a Drug Screening Search
To initiate a drug screening search, complete the following steps:
1.
From any caMOD viewer window, click the Search Models link at the top of the page.
2.
Click the Drug Screening option listed among the Search options in the upper-left of the browser window.
3.
On the Drug Screening Data form, enter the appropriate information. (Fields are described in the following table .) Note that you can check all of the check boxes in one search. Items with an asterisk are required fields.
 
Drug Screening Fields
Description
*NSC Number
Enter the NSC number in the text box. Once you start entering the number, caMOD autocompletes matching NSC numbers from the caMOD database.The NSC number is a unique drug identifier issued by the Nomenclature Standards Committee (NSC) of the U.S Food and Drug Administration.
•
If you don't know the number, click the Find NSC# button. The button opens a page at the Developmental Therapeutics Program (DTP) at NCI which allows you to find the NSC number for a specific drug.
•
On the DTP page, select Chemical name starts or Chemical name number. (By default, the radio button is set to find the NSC number.)
•
Enter text in the Search Parameters field, and click Submit.
•
Once you have found the NSC number, copy the number from the field onto the corresponding field on the caMOD Drug Screening page.
DTP Yeast Screening Data
Check the box to find yeast screening data related to the specified drug.
DTP In-vivo Screening Data
Check the box to find in-vivo screening data related to the specified drug.
Pre-clinical Trials
Check the box to find pre-clinical trials with this compound or drug related to the specified drug.
Clinical Trials
Check the box to find clinical trials with this compound or drug related to the specified drug.
4.
5.
Click Submit to initiate the search.
6.
Click Reset to return the form to default settings: If you are entering data for he first time, Reset clears the form (resets it to its defaults). If you are changing search parameters, Reset returns the form back to the unmodified state (the default).
