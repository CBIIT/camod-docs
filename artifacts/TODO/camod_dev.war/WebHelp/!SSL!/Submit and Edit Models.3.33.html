Viewing MrSID Images
MrSid images are high-quality images that have been compressed. The images are shown in the MrSid image viewer. The software allows the user to zoom in and out, move the pan to the left, right, up and down, or show the complete image (full size view).
Depending on the selection made by the user the specific part of the image will be downloaded from the server. This might take a moment so be patient.
