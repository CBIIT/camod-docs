Search Query History
To see a Search Query History, click the View Query History hypertext link in the lower left sidebar. (This link displays in the left sidebar in any caMOD viewer window.)
The Query History that opens displays Query Names for the last 20 searches you have launched. It also displays the time point of the search and the number of hits in the search. By clicking the associated RUN QUERY button, you can resubmit the query.
Search results re-display in a Search Results page. If the search was launched as a Keyword Search, the query term is highlighted in yellow wherever it occurs in the search results text.
