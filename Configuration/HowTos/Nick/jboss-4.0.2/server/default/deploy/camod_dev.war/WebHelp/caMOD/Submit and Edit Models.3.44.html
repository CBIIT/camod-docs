Investigator
This form displays for you to enter model availability information for an animal model.
If the Investigator is willing to send the mice to other laboratories, follow these steps:
1.
Enter the Strain Name.
2.
Select the name of the [Principal Investigator] from the drop-down list.
Scientists who would like to request the mice from the lab can directly contact the PI.
For more information about filling out the form, see Model Availability.
