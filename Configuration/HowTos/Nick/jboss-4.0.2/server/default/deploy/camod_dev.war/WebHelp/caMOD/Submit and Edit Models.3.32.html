Submitting and Editing Models > Publications
Publications
To enter or edit the information about publications:
1.
Click Publications from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Publication red hypertext link. This opens the Publications form.
3.
Enter or select the appropriate information about publications related to your animal model in the text entry fields. Items with an asterisk are required fields.
Publications Fields
Description
*First Author
Primary author for the publication
*Publication Status
Select Published or Unpublished. Unpublished can include work that is submitted or in preparation.
*Publication where reported first time?
Is this the publication where the animal model being entered was described for the first time? Select Yes or No.
**PubMed ID (PMID)
Enter the PubMed ID (PMID) or click the link to look up the PubMed ID. Enter it in the appropriate field. Click the Fill In Fields button. caMOD automatically retrieves the publication information based on the PubMed ID and populates the Publications form fields identified in this table with bullets.
 
Note: For books and journals not listed in PubMed, enter the appropriate information on this form.
ZFin Number
This field displays only if the model you select is a zebrafish. Enter the zebrafish (ZFN) identifier of the modified allele. Note: You can copy and paste an entire ZFIN ID into this field.
Click Find ZFIN ID to open the ZFIN website where you can learn more about the zebrafish, including the ZFIN ID for your model.
J Number
This field displays if you enter any animal model other than a zebrafish. J Number is an internal identifier for a publication used by the Jackson Laboratory. Enter this internal identifier in the appropriate text field. Click the Find J# button to open the MGI Reference Query form where you can enter the identifying information to find the J Number.

4.
Click the Submit button to send the publication to the database.
5.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
**Either the PMID or Title of Publication (not both) is a required field.
