In-Vivo Drug Screening Search Results
You can launch a search for in-vivo drug screening compounds. The results initially display on a In-Vivo Drug Screening Search Results Summary Page. From the Summary Page, you can click the hypertext link to open a In-Vivo Drug Screening Search Results Details Page associated with each hit.
