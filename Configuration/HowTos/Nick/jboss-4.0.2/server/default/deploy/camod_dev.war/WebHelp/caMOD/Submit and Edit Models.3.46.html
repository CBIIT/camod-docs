ZFIN
This form displays for you to enter model availability information for a zebrafish.
If the model is available from the ZFIN database, follow this steps:
1.
Enter the Strain Name (required) and ZFIN Identifier.
2.
Click the Find Info button to open the ZFIN website (http://zebrafish.org) where you can find the appropriate information for your model.
Models can be purchased from Zirc, which is the commercial arm of ZFIN (http://zebrafish.org/zirc/home/guide/php).
Note:
You must enter the stock number as shown here: ZDB-GENO-070209-223.
For more information about filling out the form, see Model Availability.
