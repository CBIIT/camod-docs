Carcinogenic Interventions
Carcinogenic interventions capture information about changes in the animals that are not heritable. In contrast, Genetic Description captures information about germ line mutations (which can be inherited by the next generation).
Data submitted to the carcinogenic interventions pages describe treatments that are designed to trigger or accelerate tumor development. All treatments intended to prevent, slow down, or otherwise treat cancerous growth should be submitted as therapeutic experiments. See Therapy for more information.
