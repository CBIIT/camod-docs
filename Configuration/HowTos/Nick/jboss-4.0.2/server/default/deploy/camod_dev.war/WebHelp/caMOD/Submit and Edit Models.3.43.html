MMHCC Repository
This form displays for you to enter model availability information for an animal model.
The Mouse Models of Human Cancers Consortium (MMHCC) Repository is an NCI-funded resource for mouse cancer models and associated strains. The repository makes strains available to all members of the scientific community.
To inquire about the availability of the model from the MMHCC Repository, follow this step:
1.
Enter the Strain Name and Strain Number.
2.
Click the Find Info button to open the MMHCC Repository website (http://mopuse.ncifcrf.gov) to find further information.
For more information about filling out the form, see Model Availability.
