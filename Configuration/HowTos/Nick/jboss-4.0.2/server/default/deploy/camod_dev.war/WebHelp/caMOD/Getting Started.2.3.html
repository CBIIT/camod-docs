caMOD Data
Starting with caMOD 2.4, the data set includes data from the Mouse Tumor Biology Database (MTB). The data were courteously provided by our colleagues of the MTB team. Our special thanks go to Dr. Janan Eppig, Debbie Krupke, and Matthew Vincent. The Mouse Tumor Biology Database is part of the Mouse Genome Informatics program at the Jackson Laboratory, Bar Harbor, Maine. It can be found on the World Wide Web at http://www.informatics.jax.org.
