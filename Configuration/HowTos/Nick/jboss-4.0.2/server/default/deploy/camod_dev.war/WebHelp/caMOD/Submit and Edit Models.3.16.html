Induced Mutation
The "induced mutation" page gathers information about mutations that were triggered by chemicals, radiation or other types of mutagenes and that become hereditary.
To enter or edit the Induced Mutation information for an animal model:
1.
Click Genetic Description from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Induced Mutation red hypertext link. This opens the Induced Mutation form.
3.
Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.
Induced Mutation Text Fields
Description
*Name of Inducing Agent
Enter the name of the agent inducing the mutation.
*Inducing Agent Category
Inducing agent is that which caused the mutation. Select the agent category from the drop-down list or select Other and enter an agent category not found in the drop-down list in the following text box.
Other Category
Enter the Induced Mutation Category that is not listed in the drop-down list.
Note: This field is only writable after you select Other in the previous field.
CAS Number
Enter the Chemical Abstracts Service (CAS) number for a chemical or drug. Use the provided hypertext link (Find CAS#) to find the CAS number. Copy the number and paste into the field on the caMOD Chemical/Drug page.
Note: You can also retrieve the CAS number from the DTP page described in the previous row of this table. It will display with the NSC number on the Chemical/Drug page.
Gene ID (Entrez)
Enter the Gene Entrez ID number. Click the Find ID button for a link to Entrez where you can locate the ID.
Description
Describe in the text field how the mutation was induced.
Observation
Note the genetic mutations that were observed.
Method of Observation
Explain how you learned about the gene(s) that were modified? Note: This field becomes writable only after you have entered information in the Observation field.
ZFIN Identifier
This field displays only if the model you select is a zebrafish. Enter the zebrafish (ZFN) identifier of the modified allele. Note: You can copy and paste an entire ZFIN ID into this field
Click Find ZFIN ID to open the ZFIN website where you can learn more about the zebrafish, including the ZFIN ID for your model.
MGI Number
This field displays only if the model you select is a mouse. Enter the Mouse Genome Informatics (MGI) ID for this mutation. Note: You can copy and paste the entire MGI ID (including “MGI”) into this field.
Click Find MGI ID to open the MGI website where you can learn more about the MGI, including the MGI ID for your model.
RGD Identifier
This field displays only if the model you select is a rat. Enter the Rat Genome Databe (RGD) ID for this mutation. Note: You can copy and paste the entire RGD ID (including “RGD”) into this field.
Click Find RGD ID to open the RGD website where you can learn more about the RGD, including the RGD ID for your model.
Comment
Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.
4.
Click the Submit button to execute the submission of the model information.
5.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
