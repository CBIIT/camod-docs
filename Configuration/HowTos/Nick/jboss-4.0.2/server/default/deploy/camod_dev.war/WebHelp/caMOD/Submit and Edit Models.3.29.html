Morpholinos
This category is included in caMOD for working with zebrafish. The primary tools for such work are “morpholinos”. Morpholinos, or morpholino phosphorodiamidate oligonucleotides (MOs), are small antisense molecules (about 25 nucleotides) that are used to block regions of RNA and thus knock down a gene function. MOs are the main tool for reverse genetics in fish models.
To enter or edit the morpholino information for an animal model:
1.
Click Transient Interference from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Morpholino red hypertext link. This opens the Morpholino form.
3.
Enter or select the appropriate information about your morpholino in the text entry fields. Items with an asterisk are required fields.
Morpholino Fields
Description
Source
Select the source of the morpholino in the drop-down menu. Options are companies that make morpholinos: Gene Tool; Biosystem
If the appropriate source is not listed, select Other from the drop-down list and enter the Other Source in the corresponding text field.
Other Source
Enter the morpholino ource that is not listed in the drop-down list. Note: This field is only writable after you select Other in the previous field.
Type
Select from the drop-down menu the morpholino type, Custom or Standard.
Sequence Direction
Select the sequence direction in the drop-down menu: 5' - 3' or 3' - 5'.
*Targeted Region
Enter the Targeted Region (gene or gene product to be knocked down) affected by the morpholino in the text box.
Note: After the data have been submitted, the morpholino is identified under the Transient Interference > Morpholino category by the term used in this text box.
Concentration
Enter the Concentration of the sample in value and measurement units.
Delivery Method
Select from the drop-down menu the method used to deliver the MOs. If the appropriate method is not listed, select Other from the drop-down list and enter the Other Delivery Method in the corresponding text field.
Other Delivery Method
Enter Other Delivery Method that is not listed in the drop-down list.
Target Site
Select the region within the gene you target (start site or splice site).
Visualization Ligands
Since zebrafish embryos are clear, MOs are combined with fluorescent ligands for visualization purposes. For example, fluorescein isothiocyanate is a visualization ligand.
Other Visualization Ligands
Enter Other Visualization Ligand that is not listed in the drop-down list.
Comment
Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.
4.
Click the Submit button to execute the submission of the model information.
5.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
