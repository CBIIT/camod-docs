Submitting and Editing Models > Genetic Description > Genomic Segment
Genomic Segment
To enter or edit the genomic segment information for an animal model:
1.
Click Genetic Description from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Genomic Segment red hypertext link. This opens the Genomic Segment form.
3.
Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.
Genomic Segment Text Fields
Description
*Integration
Location of Integration
Select the Random or Targeted radio button, indicating whether the integration of the genomic segment was random or targeted.
If you selected Targeted [Integration], enter the location of integration of the genomic segment into the genome (i.e. chromosome locus).
*Segment Type
Select the Segment Type from the drop-down list: Cosmid, YAC, BAC or Plasmid;
If the appropriate Segment Type is not listed, select Other from the drop-down list and enter the Other Segment Type in the text field under the list.
Other Segment Type
Enter Segment Type that is not listed in the drop-down list.
Note: This field is only writable after you select Other in the previous field.
Segment Size
Enter appropriate segment size information.
*Designator
Clone Designator: Indicate the location of the clone in the library, for example, RPCI Human Male Library, 24F11.
Comment
Enter any information that did not fit or was not suitable for other fields on the page.
ZFIN Identifier
This field displays only if the model you select is a zebrafish. Enter the zebrafish (ZFN) identifier of the modified allele. Note: You can copy and paste an entire ZFIN ID into this field.
Click Find ZFIN ID to open the ZFIN website where you can learn more about the zebrafish, including the ZFIN ID for your model.
MGI Number
This field displays only if the model you select is a mouse. Enter the Mouse Genome Informatics (MGI) ID for this mutation. Note: You can copy and paste the entire MGI ID (including “MGI”) into this field.
Click Find MGI ID to open the MGI website where you can learn more about the MGI, including the MGI ID for your model.
RGD Identifier
This field displays only if the model you select is a rat. Enter the Rat Genome Databe (RGD) ID for this mutation. Note: You can copy and paste the entire RGD ID (including “RGD”) into this field.
Click Find RGD ID to open the RGD website where you can learn more about the RGD, including the RGD ID for your model.
4.
Under the gray line in the Genomic Segment form, select or enter the requested information about the model construct, if you decide to upload a map of the construct. For more information, see Model Construct.
5.
Click the Submit button to execute the submission of the model information.
6.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
