Jackson Lab
This form displays for you to enter model availability information for an animal model.
 
Jackson Lab is a large international animal models distributor.
If the model is available from the Jackson Laboratory, follow these steps:
1.
Enter the *Strain Name and Stock Number of the model in the appropriate text field.
2.
Click the Find Info button to open the Jackson Lab website (http://jaxmice.jax.org) where you can find the appropriate information for your model.
For more information about filling out the form, see Model Availability.
