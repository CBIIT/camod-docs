Spontaneous Mutation
To enter or edit the Spontaneous Mutation information for an animal model:
1.
Click Genetic Description from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.
2.
Click the Enter Spontaneous Mutation red hypertext link. This opens the Spontaneous Mutation form.
3.
Enter or select the appropriate information about your animal model in the text entry fields.
Spontaneous Mutation Text Fields
Description
Gene Name
Enter the name of the gene in which the spontaneous mutation occurred.
Observation
Note the genetic changes that were observed.
Method of Observation
Specify the method(s) used to discover the genetic changes. Include the technology and the tests that were used. Note: This field becomes writable only after you have entered information in the Observation field.
Gene ID
Enter the Gene Entrez ID number. Click the Find ID button for a link to Entrez where you can locate the ID.
ZFIN Identifier
This field displays only if the model you select is a zebrafish. Enter the zebrafish (ZFN) identifier of the modified allele. Note: You can copy and paste an entire ZFIN ID into this field.
Click Find ZFIN ID to open the ZFIN website1 where you can learn more about the zebrafish, including the ZFIN ID for your model.
MGI Number
This field displays only if the model you select is a mouse. Enter the Mouse Genome Informatics (MGI) ID for this mutation. Note: You can copy and paste the entire MGI ID (including “MGI”) into this field.
Click Find MGI ID to open the MGI website where you can learn more about the MGI, including the MGI ID for your model.
RGD Identifier
This field displays only if the model you select is a rat. Enter the Rat Genome Databe (RGD) ID for this mutation. Note: You can copy and paste the entire RGD ID (including “RGD”) into this field.
Click Find RGD ID to open the RGD website where you can learn more about the RGD, including the RGD ID for your model.
Comment
Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.
1
ZFIN is the Zebrafish Model Organism database.

4.
Click the Submit button to execute the submission of the model information.
5.
Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).
