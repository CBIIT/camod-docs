IMSR
This form displays for you to enter model availability information for an animal model.
IMSR (International Mutant Strain Resource) is a massive database linking animal model databases throughout the world. In a sense, IMSR is a "one-stop shop" for obtaining animal models.
If the model is available from the IMSR, follow this steps:
1.
Enter the Strain Name and Stock Number in the text fields.
2.
Click the Find Info button to open the IMSR website (http://informatics.jax.org) to find further information.
For more information about filling out the form, see Model Availability.
