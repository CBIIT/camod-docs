Combining Search Criteria
In both Simple Search Mode and Advanced Search Mode, when multiple search criteria are entered, the search is executed by automatically combining them with an "AND" statement. This, in effect, can narrow the number of retrieved records.
Example:
•
If you launch a search for models where the model descriptor/name contains WAP, the search might return six matches.
•
If you search for models where the model descriptor/name contains WAP and the tumor site is mammary gland, the search might return less than six matches, because both search criteria need to be fulfilled in order to get matching records.
