Assigning Models to Editors
Only models that were approved by the Screener can be send to the Editor.
1.
To see models that need to be reviewed, the Coordinator clicks the View Model Assignment option in the left sidebar.
2.
In the Models Assignment dialog box, the Coordinator selects Screened - Approved models from the drop-down list.
3.
The Coordinator marks the models by clicking the check box in front of the model name.
