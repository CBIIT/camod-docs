COnfiguring oracle

1) Install

2) Bring up the Oracle Enterprise Management Console in standalone mode

3) Click on the instance name and connect as SYS with SYSDBA privledges (password set in install)

4) Click on XML Database / Configuration and set the http-port to 8081 and Apply

5) Click on Security /Users and create the CAMOD user with camod password.

6) Select the role tab and add the RESOURCE role and apply.
